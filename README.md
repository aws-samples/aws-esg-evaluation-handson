# ESG Evaluation Handson


## Content

* 気候変動に対する企業の取り組みを評価する業務を自然言語処理を用いて効率化する
   * 講義資料: [ESG評価に対する自然言語処理の活用Workshop](https://speakerdeck.com/icoxfog417/esgping-jia-nidui-suruzi-ran-yan-yu-chu-li-falsehuo-yong-workshop)
